# Contributing to The Agentic Stack

Thanks for helping make this resource better!

## How to Contribute

1. Fork this repo
2. Add your tool to the appropriate layer in `README.md`
3. Follow the format: `| Tool Name | One-line description | [GitHub](link) |`
4. Submit a PR

## Rules

- Tool must fit one of the 4 layers (Orchestration, Agents, Tools, Guardrails)
- Must be actively maintained (last commit within 6 months)
- No duplicates
- No paid-only tools without a free tier

## Suggesting a New Layer or Category

Open an issue with the label `enhancement` to propose structural changes.
